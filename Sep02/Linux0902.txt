
Ветка  GIT — это снимок всех входящий в репозиторий файлов по состоянию на какой-то момент времени
SSH ключи нужны для авторизации на GITHUB т. к. с 18 августа 2022 года авторизация на сервере под паролем уже не осуществляется.
Сервис GitHub Pages предоставляет отображение статичных web страниц формирующихся из файлов пользователей, которые разместили их на ресурсе  GITHUB.

Создание ветки локально:
git branch name

удаление ветки:
git -D branch name
 
переключение между ветками:
git checkout name


загрузка веток на GitHub:
git push -u origin name

удаление веток на GitHub:
git push -D origin name
