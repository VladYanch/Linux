Linux 15.09

(1) Зачем предназначен файл .gitignore? для создания списка файлов невключаемых в git
(2) Что делает команда git remote rm origin? отвязывает текущую папку от GitHub
(3) В каких случаях происходит MERGE CONFLICT? В случае, когда с одном из двух git-ов не было окончательного коммита
(4) Как посмотреть отличия между ветками? git diff vetka1 vetka2

(5) Какой командой удалить локальную и удаленную ветки?
Локально
git -d branch vetka

удаление веток на GitHub:
git push -d origin vetka

(6) Как сделать файл неотслеживаемым (untracked)? git reset file_name
